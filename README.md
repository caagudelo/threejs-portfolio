# Three.js Portfolio

Este proyecto es un portafolio web moderno creado con Vite, React y Three.js. Incluye una escena 3D de ejemplo y está listo para ser desplegado como sitio estático en cualquier hosting compartido.

## Scripts

- `npm run dev` — Ejecuta el servidor de desarrollo
- `npm run build` — Genera la versión de producción en la carpeta `dist`
- `npm run preview` — Previsualiza la versión de producción

## Despliegue en hosting compartido
1. Ejecuta `npm run build`.
2. Sube el contenido de la carpeta `dist` a tu hosting compartido.

## Ejemplo de integración Three.js
La página principal muestra un cubo 3D animado usando Three.js.

---

Personaliza la escena 3D en `src/App.jsx` para destacar tus proyectos o información personal.
